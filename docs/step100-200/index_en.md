# STEP100/200

## Overview

STEP100/200 is a completely DIY style motor driver where you collect and assemble the all modules yourself.

| Model | Motor Driver | Motors | In a ward... |
| --- | --- | --- | --- |
| STEP100 | PowerSTEP01 | 1 | Small STEP400 |
| STEP200 | L6470 | 2 | Small STEP800 |

[The firmware](https://github.com/ponoor/step-series-universal-firmware) is provided as an open source project.

This document describes mainly how to assemble and how to write firmware. For the actual usage, please refer to the [STEP400/800 documents](https://ponoor.com/en/docs/step-series/) and replace the MotorID range as appropriate as shown below.

| Model | Approproate Product | Range of MotorID |
| --- | --- | --- |
| STEP100 | STEP400 | 1, 255 |
| STEP200 | STEP800 | 1-2, 255 |

The other differences are explained in the [usage](https://ponoor.com/en/docs/step100-200/usage/) section.
